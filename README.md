# Employee Attendance Management System

## Overview

The Employee Attendance Management System is a web-based application developed using ASP.NET and C#. It enables organizations to manage employee attendance 
efficiently by recording daily attendance, calculating working hours, generating reports, and maintaining employee records through a centralized interface.
## ✨ Features

### Employee Management
- Add new employees
- Update employee information
- Delete employee records
- View employee details

### Attendance Management
- Mark daily attendance
- Track employee working hours
- Calculate average working hours
- View monthly attendance records

### Reports
- Generate attendance reports
- View monthly summaries
- Export attendance data to Excel

### User Interface
- Clean and user-friendly interface
- Easy navigation
- Responsive data tables

## 🚀 Getting Started

### Prerequisites

Before running this project, make sure you have:

- .NET SDK
- Visual Studio Code or Visual Studio
- Git

### Clone the Repository

```bash
[git clone https://github.com/venumadhavi-4/employee-attendance-system.git]
```

### Navigate to the Project

```bash
cd employee-attendance-system
```

### Restore Dependencies

```bash
dotnet restore
```

### Run the Application

```bash
dotnet run
```
